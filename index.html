How it works

1

Install

Add Sticky Notes from the Chrome Web Store (or load unpacked for developer mode).

2

Click & Create

Click the extension icon in your toolbar. A note appears instantly on the page.

3

Organize

Drag to reposition, resize, minimize, pick a color. Notes auto-save as you type.

4

Come Back

Notes are tied to each page URL. Revisit a site and your notes are right where you left them.

Built for developers too

Clean modular architecture, open source, MIT licensed

📦 Modular Codebase

v2 split one 740-line file into 7 focused modules — ui.js, drag.js, storage.js, dashboard.js, features.js, config.js, and content.js.

🛡️ Manifest V3

Modern Chrome Extension standard with service worker, content scripts, and strict CSP compliance.

⚡ Build Pipeline

Automated CI/CD via GitHub Actions. Bundles ES modules into a single IIFE for maximum compatibility.
